# 栃木県企業向け無料診断ツール

栃木県の企業様向けに、業界別の課題をヒアリングし、診断結果と改善提案を提供する無料診断ツールです。

## 📋 プロジェクト概要

このツールは、企業が直面している課題を「DX・IT化」「業務効率化」「売上向上」「コスト削減」の4つの観点から診断し、具体的な改善策を提案します。

### 主な機能

1. **業界選択**
   - 6つの業界に対応（教育、医療・福祉、アパレル・美容、運輸・物流、広告マスコミ、人材）
   - 各業界に最適化された診断内容

2. **診断質問（12問）**
   - 4つのカテゴリ × 3問ずつ
   - **困りごとベースの質問**（実際に解決できる課題を聞く）
   - 5段階評価（1: 全く当てはまらない 〜 5: 非常に当てはまる）
   - 質問の前後移動が可能
   - **全業界共通の質問**で横断的に診断

3. **診断結果表示**
   - 総合スコア（60点満点）とパーセンテージ表示
   - レーダーチャートによるカテゴリ別分析
   - カテゴリごとの詳細スコアとステータス
   - スコアに応じた具体的な解決策の提案
   - **「私ができること」セクション** - 改善が必要な上位2カテゴリに対する具体的なサポート内容を表示

4. **問い合わせ導線**
   - Timerex予約フォームへのリンク
   - 診断結果に基づいた無料相談への誘導

### 診断カテゴリと質問内容

#### 1. **DX・IT化**（デジタル活用・仕組みづくり）
   - 情報共有や連絡業務に時間がかかりすぎていませんか？
   - 顧客情報や予約管理が煩雑になっていませんか？
   - ITツールを導入しても、社内で使いこなせていない状況はありませんか？

#### 2. **業務効率化**（ムダ削減・属人化の解消）
   - 同じ作業を何度も繰り返していたり、手作業が多すぎませんか？
   - 特定の人しかできない業務があり、属人化していませんか？
   - 会議や報告書作成など、成果に直結しない業務に時間を取られていませんか？

#### 3. **売上向上**（集客・成約率・リピート改善）
   - 集客が不安定で、紹介や口コミ頼みになっていませんか？
   - ホームページやSNSで、何を発信すれば良いかわからない状況ですか？
   - リピーターが増えず、新規顧客の獲得ばかりに追われていませんか？

#### 4. **コスト削減**（固定費・ムダな支出の最適化）
   - 何にいくら使っているか、正確に把握できていますか？
   - 昔からの契約や外注をそのまま続けていませんか？
   - コストを削減したいが、どこから手をつければ良いかわからない状況ですか？

## 💡 使い方

### 基本的な流れ

1. **業界を選択**
   - トップ画面で貴社の業界を選択します

2. **質問に回答**
   - 12問の質問に5段階で評価します
   - 前へ/次へボタンで質問を移動できます

3. **診断結果を確認**
   - 総合スコアとカテゴリ別分析を確認
   - 具体的な改善策を確認
   - 「私ができること」で具体的なサポート内容を確認

4. **無料相談を申し込む**
   - より詳しいサポートが必要な場合はTimerex予約フォームから申し込み
   - 新しいタブで予約ページが開きます

### スコア評価基準（困りごとベース）

**質問は「困っているか」を聞くため、スコアが高い = 困っている = 改善が必要**

- **70%以上（要改善）**: 課題を多く抱えている状態 → 重点的な改善提案
- **40-69%（改善の余地あり）**: 一部に課題がある状態 → 具体的な改善施策を提案
- **39%以下（良好）**: 比較的問題が少ない状態 → さらなる向上のための提案

## 🔧 カスタマイズ方法

### 質問内容の変更

`js/questions.js` ファイルの `commonQuestions` オブジェクトを編集します（全業界共通）：

```javascript
commonQuestions.dx.questions[0] = '新しい質問内容';
```

### 解決策の変更

各業界・カテゴリの `solutions` オブジェクトを編集します：

```javascript
solutions: {
    low: '良好な状態時の解決策',
    medium: '改善の余地あり時の解決策',
    high: '要改善時の解決策'  // 困りごとベースなので high = 要改善
}
```

### 「私ができること」の変更

`myServices` オブジェクトで、各カテゴリのサポート内容を編集できます：

```javascript
myServices.dx = {
    title: 'サービスタイトル',
    problems: ['課題1', '課題2', '課題3'],
    solutions: ['サポート内容1', 'サポート内容2', ...],
    message: 'メッセージ'
}
```

### 業界の追加

1. `js/questions.js` に新しい業界データを追加
2. `index.html` に業界選択カードを追加
3. 適切なアイコンを設定

### デザインの変更

`css/style.css` で以下の要素をカスタマイズできます：

- カラースキーム（グラデーション、ボタン色など）
- フォントサイズとスタイル
- レイアウト（グリッド、余白など）
- アニメーション効果
